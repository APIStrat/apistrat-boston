The time has come to open up ticket sales for API Days - APIStrat Berlin, this April 24th and 25th. This is the first round of early-bird tickets available for the joint API industry conference put on by both API Days and APIStrat in Germany this spring.

As we have done with past events, we will be rolling out tickets in waves, but if you want to get the best deal for ticketsnow is the time. This version of the conference will only be two days, and seats will be much more limited than before, so make sure you get in on the action early, and get registered now.

There are other sponsorship opportunities as well, so if you company wants to increase your participation in the conversation, make sure and contact us now. We are also closing the request for talks, so if you have a last minute proposalsubmit now!!
