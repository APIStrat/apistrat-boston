The APIStrat Chicago edition is just a couple of months away. Today we're pleased to announce the preliminary program so you can start planning which sessions to head for and get a sneak peek at some of the exciting speakers we have lined up.

There are a number of slots that we're still waiting to announce, and we'll be releasing details on them over the next few weeks. In the meantime, we're looking forward to our best APIStrat ever, featuring topics including:

  • API Architecture
  • API Testing
  • Internal APIs
  • API Design
  • And many more

Check out the preliminary APIStrat Chicago program now.

APIStrat Chicago will take place at the Chicago Hyatt Regency, at the city's center. Tickets are selling fast so grab yours while you still can. Take advantage of early bird prices now (ends August 8) for $200 off general admission price.
