As the API economy gathers steam, the API Strategy & Practice Conference today announced its next North American event will be held in Austin, Texas, November 18-20, 2015.

The conference is co-organized by 3Scale and API Evangelist and is dedicated to sharing, informing and learning about the rapidly growing API industry and marketplace. The event will include talks, panels and keynotes with over 80 speakers covering diverse topics from API security and design, to business models, operations and marketing.

As of today the call for speakers is open: http://austin2015.apistrat.com/call-for-papers/. 

“APIStrat events always impress me with the quality of insight from speakers as the API world is moving fast and seeing best practices from so many companies and individuals is extremely valuable,” said Steve Willmott, CEO of 3Scale.

Kin Lane, of API Evangelist adds: “After five years of working in the space, APIStrat is the place where the community gathers to have the important conversations that move the API economy forward.  In 2015 it is not just about REST and API management, so make sure you are present, and are in the know.”

For this edition, the event has expanded the team to reach greater heights and has added Mark Boyd as the program chair and Keith Casey of as the local event chair.

Mark is veteran journalist in the API field writing for ProgrammableWeb and The New Stack: “As a writer documenting best practice examples and new knowledge in the API economy, I have always appreciated API Strategy & Practice as a great source for new contacts, emerging ideas and proven examples of how APIs are making a difference to businesses and communities. As program chair, I'm looking forward to adding to the successes of APIStrat's previous conferences by helping build an exciting program that highlights both leading API experts and practitioners.”

Keith Casey, one of the organizers of the Austin API Meetup http://austinapi.com/, said, "APIStrat is one of the few events where the doers, planners and thinkers come together and discuss the ideas and technologies that are driving us forward. Time and time again, five minute conversations and 30 minute sessions have saved us weeks of development."

The 2015 API Strategy and Practice Conference will be held at the Sheraton Hotel at the Capitol in Austin and provide a high quality environment for presentations, sessions and networking. The event will also feature a sponsor’s area, hack-zone and themed API tutorial sessions.

The API Strategy & Practice conference takes the comfort, safety and inclusiveness of everyone involved in the event very seriously. All attendees are required to adhere to a code of conduct to insure that everyone is free to enjoy the collaborative environment the conference strives to foster. Additionally, the event is vendor neutral, meaning topics are not product or brand-centric.

To learn more about the conference and to pre-register please visit http://apistrategyconference.com

Media that would like to attend can secure complimentary press passes by contacting apiconference@3scale.net
