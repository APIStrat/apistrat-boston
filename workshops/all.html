Register for: A Modern API Lifecycle Workshop with Kin Lane

The API industry has significantly evolved since its early days, moving well beyond just deploying and managing APIs. Many API providers are getting sophisticated in how they approach everything from API design, to addressing monitoring, testing, and security. Join Kin Lane, the API Evangelist for a walk through the 30 most common areas top API providers are working on, across the entire lifecycle of their APIs. Bring your laptops, as this workshop will give you a wealth of resources you can explore across the modern API lifecycle.

Register for: Focus on API Workflow Workshop with Tony Tam

APIs are core to your business but are often treated more like artifacts of code rather than first class citizens in the development stack. In this talk, we’ll focus on the API-First process with Swagger and how open-source tools can speed things up and can make a more cohesive process for the entire development team.
