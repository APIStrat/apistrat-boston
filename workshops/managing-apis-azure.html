Workshop: Managing APIs with Azure API Management

Hosted by:

Instructors:

Vladimir Vinogradsky

Miao Jiang

Darrel Miller

Description:


Learn how to abstract, control, and analyze APIs using Azure API Management.

We will start with giving you a brief overview of Azure API Management. Then we will demonstrate the key features and capabilities of the product – policies and expressions, built-in and custom analytics, configuration over Git, portal customization and more.

To follow along you need to provision your own instance of Azure API Management and have it ready by the start of the session. See https://aka.ms/createapim for directions.
